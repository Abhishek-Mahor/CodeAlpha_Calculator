# Basic Calculator Project

This project is a simple **Basic Calculator** built during my internship with **CodeAlpha**.  
It performs fundamental arithmetic operations such as addition, subtraction, multiplication, and division, with a clean and user-friendly interface.

---

##  Project Structure
- **index.html** → Main HTML file containing the calculator layout.
- **style.css** → Stylesheet for designing the calculator interface.
- **app.js** → JavaScript logic to handle operations and user interactions.

---

##  Features
- Perform basic arithmetic: Addition, Subtraction,  Multiplication,  Division.
- Responsive and minimal design.
- Error handling for invalid inputs.
- Clear button to reset calculations.

---

## How to Run
1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
3. Start performing calculations instantly.

---

##  Technologies Used
- **HTML5** → Structure
- **CSS3** → Styling
- **JavaScript (ES6)** → Functionality
